# 🔐 Advanced Encryption & Decryption Tool

## 📌 Project Overview
This project is a GUI-based Encryption & Decryption Tool developed using Python and Tkinter as part of the DecodeLabs Cyber Security Internship Project 2.

The application uses the Caesar Cipher algorithm to encrypt and decrypt user messages securely.

---

# 🚀 Features

✅ GUI-based desktop application  
✅ Encrypt messages using Caesar Cipher  
✅ Decrypt encrypted messages  
✅ User-friendly interface  
✅ Error handling and input validation  
✅ Dark-themed professional UI  
✅ Clear button functionality  

---

# 🛠 Technologies Used

- Python
- Tkinter GUI Library
